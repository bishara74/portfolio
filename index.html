Featured Project

GPU Training Benchmark

Quantified the performance gains of GPU acceleration by benchmarking PyTorch model training. Achieved a 13x speedup (23.7 min on CPU vs. 1.8 min on GPU). Implemented GPU resource management by explicitly moving tensors and data batches to the CUDA device. Published as a community tutorial/notebook.

Python • PyTorch • CUDA • Jupyter Notebook
